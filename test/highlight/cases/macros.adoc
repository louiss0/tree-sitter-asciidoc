== Math Macros

The quadratic formula is stem:[x = (-b +- sqrt(b^2-4ac))/(2a)].

LaTeX inline: latexmath:[\sum_{i=1}^n x_i = x_1 + x_2 + \cdots + x_n]

AsciiMath inline: asciimath:[sum_(i=1)^n x_i]

== Footnotes

This text has a footnote.footnote:[This is the footnote text]

You can also reference footnotes.footnote:ref1[Referenced footnote]

Later reference: footnoteref:ref1[]

== Index Terms

This sentence contains an ((index term)) that will be indexed.

There's also (((hidden index term))) that won't appear in text.

You can use indexterm:[primary] and indexterm2:[visible term] macros.

Advanced indexing: indexterm:[Primary, Secondary, Tertiary]

== Passthrough Macros

Raw HTML: ++++<span class="special">HTML content</span>++++

Pass macro: pass:[<u>underlined</u>] text.

Substitution control: pass:q[*bold* _italic_] with quotes only.
